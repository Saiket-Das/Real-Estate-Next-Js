# Real Estate

### Live site
https://real-estate-next-js-nine.vercel.app/

Real Estate is a frontend web application made using Next JS.


### Dependencies Used
* Chakra UI
* axios
* millify 
* react-icons 
* nprogress 
* Real estate Api - Rapid API (Bayut)
